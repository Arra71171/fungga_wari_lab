# Phase 6 ‚Äî Plan 05: Update Seed Data to Use Portrait Illustrations

## Context
The Convex seed data in `convex/seed.ts` uses Unsplash landscape photo URLs for
`coverImageUrl`. These must be replaced with the local portrait illustration paths.
A one-time update mutation should also patch the live database to match.

---

```xml
<task type="auto">
  <name>Update seed data and live database coverImageUrl to portrait illustrations</name>
  <files>convex/seed.ts</files>
  <action>
    STEP 1 ‚Äî Update convex/seed.ts:

    Find the `coverImageUrl` values in the two `ctx.db.insert("stories", ...)` calls:

    Story 1 (Tiger and the Golden Pheasant) ‚Äî CHANGE:
    CURRENT:
    ```ts
    coverImageUrl: "https://images.unsplash.com/photo-1571217666993-...",
    ```
    REPLACE WITH:
    ```ts
    coverImageUrl: "/illustrations/tiger_pheasant.png",
    ```

    Story 2 (Khamba and Thoibi) ‚Äî CHANGE:
    CURRENT:
    ```ts
    coverImageUrl: "https://images.unsplash.com/photo-1605806616949-...",
    ```
    REPLACE WITH:
    ```ts
    coverImageUrl: "/illustrations/khamba_thoibi.png",
    ```

    STEP 2 ‚Äî Add a new patch mutation at the bottom of seed.ts to update
    any existing live documents in the database:

    ```ts
    export const patchIllustrations = mutation({
      args: {},
      handler: async (ctx) => {
        const stories = await ctx.db.query("stories").collect();
        for (const story of stories) {
          if (story.title === "The Tiger and the Golden Pheasant") {
            await ctx.db.patch(story._id, {
              coverImageUrl: "/illustrations/tiger_pheasant.png",
            });
          } else if (story.title === "Khamba and Thoibi") {
            await ctx.db.patch(story._id, {
              coverImageUrl: "/illustrations/khamba_thoibi.png",
            });
          } else if (story.title === "Sandrembi and Chaisra") {
            await ctx.db.patch(story._id, {
              coverImageUrl: "/illustrations/sandrembi_chaisra.png",
            });
          }
        }
        return "Illustrations patched successfully";
      },
    });
    ```

    Then run the patch via Convex MCP or `npx convex run seed:patchIllustrations`.
  </action>
  <verify>
    1. `pnpm run typecheck` passes
    2. Run `patchIllustrations` mutation via Convex MCP
    3. Query all stories via Convex MCP ‚Äî each story's `coverImageUrl` points to
       `/illustrations/{slug}.png`
    4. Story cards at http://localhost:3001/stories show the portrait illustrations
  </verify>
  <done>
    All stories in the database have coverImageUrl pointing to local portrait illustrations.
    The seed script also reflects the correct paths for future re-seeding.
  </done>
</task>
```
