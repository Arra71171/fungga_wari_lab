# Phase 6 — Plan 03: Refactor StoryCenter.tsx to Use PortraitFrame

## Context
`apps/web/components/story/StoryCenter.tsx` renders scene illustrations inside a
fixed-height `h-[460px]` landscape-biased container with a manual blur layer.
`StoryCanvas.tsx` already uses `PortraitFrame` correctly. `StoryCenter` should
match the same canonical approach for consistency across all reader routes.
