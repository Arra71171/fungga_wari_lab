# Phase 6 — Plan 01: Fix Shared `StoryCard` Component (Portrait Aspect Ratio)

## Context
The shared `StoryCard` in `packages/ui/src/components/StoryCard.tsx` uses `aspect-video`
(16:9 landscape) for cover image containers. Both the dashboard story list AND any consumer
of `@workspace/ui/components/StoryCard` inherits this broken orientation. This is the
highest-priority fix since it ripples across both apps.

---

```xml
<task type="auto">
  <name>Fix StoryCard shared component — aspect-video → aspect-[3/4]</name>
  <files>packages/ui/src/components/StoryCard.tsx</files>

  <action>
    In `packages/ui/src/components/StoryCard.tsx`, locate the cover image container div:

    CURRENT (line ~58):
    ```tsx

    ```

    CHANGE TO:
    ```tsx

    from landscape 16:9 to portrait 3:4. The compact variant retains its `h-full w-1/3`
